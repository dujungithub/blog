### 父级往子级页面（模板）的数据传递

我们通常会在页面之间进行跳转、重定向的操作。**这时候，我们可以选择将部分数据放在** `**url**` **里面，并在新页面** `**onLoad**` **的时候进行初始化。**

```
pageC.js
...
// Navigate
wx.navigateTo({
  url: '../pageD/pageD?name=raymond&gender=male',
})
 
// Redirect
wx.redirectTo({
  url: '../pageD/pageD?name=raymond&gender=male',
})
```

在 D 页面中，我们可以这样接收到到所传进来的参数：

```
// pageD.js
...
Page({
  onLoad: function(option){
    console.log(option.name + 'is' + option.gender)
    this.setData({
      option: option
    })
  }
})
```
